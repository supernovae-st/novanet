## Overview

Generative Engine Optimization (GEO) for AI answer engines.

**What is GEO:**
Optimization for AI-powered search engines (Perplexity, ChatGPT, Claude)
that generate answers rather than list links.

**Pipeline stages:**
1. **Mining**: GEOMiningRun discovers questions per locale
2. **Targeting**: Concepts target GEOSeedL10n questions
3. **Metrics**: GEOSeedMetrics tracks citation performance
4. **Optimization**: Content optimized for AI citations

**Node types:**
- GEOSeedL10n: Localized question/query (locale-specific)
- GEOSeedMetrics: Citation and visibility metrics
- GEOMiningRun: Mining job metadata

## Cypher Queries

### GEO seeds with metrics

Get question seeds with their citation metrics

```cypher
MATCH (seed:GEOSeedL10n)-[:FOR_LOCALE]->(l:Locale {key: $locale})
OPTIONAL MATCH (seed)-[:HAS_METRICS]->(m:GEOSeedMetrics)
RETURN seed.question AS question,
       seed.intent AS intent,
       m.citation_count AS citations,
       m.visibility_score AS visibility,
       m.updated_at AS lastUpdate
ORDER BY m.citation_count DESC
LIMIT 50
```

**Parameters:**
- `locale`: "fr-FR"

### Concept GEO coverage

**Parameters:**

### High-value GEO opportunities

Questions with high search intent but low citation

```cypher
MATCH (seed:GEOSeedL10n)-[:FOR_LOCALE]->(l:Locale {key: $locale})
MATCH (seed)-[:HAS_METRICS]->(m:GEOSeedMetrics)
WHERE m.search_volume > 1000 AND m.citation_count < 5
RETURN seed.question AS question,
       m.search_volume AS volume,
       m.citation_count AS currentCitations
ORDER BY m.search_volume DESC
LIMIT 20
```

**Parameters:**
- `locale`: "fr-FR"

### Recent GEO mining runs

## Notes

- Citation metrics track how often AI cites our content
- Higher visibility_score = more prominent in AI answers
