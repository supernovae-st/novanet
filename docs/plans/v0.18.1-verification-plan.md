# v0.18.1 Verification Plan

**Date**: 2026-03-10
**Type**: Deep Verification (Ultrathink Method)
**Agents**: 10 Opus 4.5 parallel agents

## Objective

Comprehensive verification of NovaNet codebase after v0.18.0 cleanup to ensure:
- Schema integrity across all layers (YAML → Cypher → TypeScript → Neo4j)
- Test coverage completeness
- Documentation alignment
- Code quality (no dead code, stale references)
- MCP server correctness

## Agent Assignments

### V1: Schema YAML Integrity
- Verify all 57 node classes have required fields
- Check realm/layer path matches YAML content
- Validate property types and constraints
- Confirm ADR-024 trait names (defined/authored/imported/generated/retrieved)

### V2: Arc Classes Completeness
- Verify all 145 arc classes are defined
- Check arc family assignments
- Validate cardinality constraints
- Confirm inverse arcs per ADR-026 tiers

### V3: TypeScript ↔ YAML Parity
- Compare generated.ts with source YAML

### V4: Rust Code Quality
- Run clippy with all warnings enabled
- Check for unused imports/functions
- Verify error handling consistency
- Audit unsafe code blocks

### V5: MCP Server Tools
- Verify all 14 tools have correct schemas

### V6: Test Coverage Gaps

### V7: Documentation Alignment

### V8: Neo4j Seed Data
