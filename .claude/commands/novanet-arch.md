---
description: Display the complete NovaNet architecture diagram in ASCII
argument-hint: [section]
---

Display the NovaNet architecture diagram showing the complete system overview.

If $ARGUMENTS is provided, focus on that specific section:
- "source" or "yaml" - Show only the Source of Truth section
- "infra" or "neo4j" - Show only the Infrastructure section
- "studio" - Show only the Studio section
- "packages" or "deps" - Show only the Packages Dependency Graph
- "flow" or "pipeline" - Show only the Data Flow diagram
- "all" or empty - Show the complete architecture (default)

Use the novanet-architecture skill to display the diagrams.
